Concept

レッスンではなく、創作の場として。

このアトリエでは、「正しい作曲法」を一方向に教えるのではなく、 受講者自身の感覚・関心・音の記憶から出発します。 楽譜を書くこと、DAWで音を編集すること、録音素材を変形すること、 アルゴリズムやAIを使うこと。そのすべてを、作品をつくるための手段として扱います。

一般的な作曲・DTMレッスンよりも、個人の創作プロセスに深く関わります。 まだ何を作りたいかわからない段階でも、すでに作品やポートフォリオを持っている段階でも、 目的に応じて内容を組み立てます。

01

創作中心

理論やソフト操作を目的化せず、作品を作るために必要な知識と技術を選びます。

02

音を聴くことから始める

音色、質感、空間、身体性、時間感覚を丁寧に扱い、音楽以前の感覚を作品へ接続します。

03

現代的な制作環境

DAW、Max/MSP、録音、AI、電子音響、映像との関係など、現在の制作環境に対応します。

Modules

学ぶ内容

固定カリキュラムではなく、目的に応じて必要なモジュールを組み合わせます。 初心者の場合は基礎から、経験者の場合は作品講評や制作計画から始めることができます。

Module 00
Listening / 聴く
音色、沈黙、空間、ノイズ、質感を分析し、自分の感覚と言語をつなげる。
Module 01
Composition / 作曲
旋律・和声・形式だけでなく、時間構造、素材、展開、コンセプトから作品を考える。
Module 02
Theory / 音楽理論
楽典、コード、和声、対位法、形式を、作曲と制作に接続する。
Module 03
Solfège / ソルフェージュ
読譜、リズム、聴音、音感、受験レベルの基礎力を整える。
Module 04
Electroacoustic / 電子音響
録音素材、声、環境音、加工、編集、空間化を用いた音響作品の制作。
Module 05
Sound & Technology
DAW、Reaper、Ableton Live、Max/MSP、MIDI、OSC、センサー、生成システム。
Module 06
AI and Music
AI生成、プロンプト、サウンド制作、作品コンセプト、既存の創作プロセスとの接続。
Module 07
Portfolio / Project
音大・芸大・海外音楽院・公募・作品提出に向けたポートフォリオと制作計画。

Policies

レッスン規約

受講について

レッスンはオンラインまたは対面で行います。 内容は受講者の目的や制作状況に応じて個別に設計されます。

初回面談について

初回30分のオンライン面談は無料です。 現在の状況、興味分野、制作内容、目標などを確認し、 適切な受講形式を提案します。

事前共有資料について

楽譜・音源・ポートフォリオなどの資料は、 レッスン前に共有していただくことができます。 Beginner Lesson では、資料の確認・質問への対応は 原則としてレッスン時間内に行います。 Advanced Lesson では、必要に応じて事前共有資料を簡易確認し、 レッスン内で扱うポイントを整理します。 ただし、詳細な添削・分析・制作指導はレッスン時間内で行います。

資料の提出期限について

Advanced Lesson で事前確認を希望される資料は、 原則としてレッスンの48時間前までに共有してください。 直前に共有された資料や分量の多い資料については、 事前確認ではなくレッスン時間内で確認します。

事前確認の範囲について

事前確認は、レッスンを効率よく進めるための簡易確認です。 レッスン外での詳細な添削、分析、制作指導を保証するものではありません。 具体的な改善点や制作方針の整理は、原則としてレッスン時間内で行います。

お支払いについて

レッスン料金は事前支払い制です。 Stripeによるクレジットカード決済、 または銀行振込をご相談いただけます。

FAQ

よくある質問

初心者でも受講できますか？

可能です。楽譜・DTM・音楽理論の経験が少ない場合でも、目的に合わせて基礎から始めます。 ただし、このアトリエでは単なる操作説明よりも、音を使って何を作るかを重視します。

受験やポートフォリオにも対応していますか？

はい。音大・芸大受験、海外音楽院、作品提出、公募、ポートフォリオ相談に対応します。 専門的な内容は Advanced Lesson として扱います。

どのくらいの頻度で受けるのがよいですか？

基礎からしっかり進めたい場合は月4回、制作ペースに合わせたい場合は月2回、 作品講評や相談中心の場合は月1回も可能です。内容と生活リズムに合わせて調整します。

単発相談はできますか？

可能です。作品講評、制作方針、ポートフォリオ、機材・ソフトの選び方など、1回のみの相談にも対応します。

どのソフトに対応していますか？

Reaper、Ableton Live、Logic Pro、Max/MSP、一般的なDAW、MIDI・OSCを使った制作などに対応します。 必要に応じてAI生成ツールや映像制作との接続も扱います。

外部スクール経由のレッスンとは何が違いますか？

外部スクールでは、一般的な作曲・DTMレッスンとして受講しやすい形になっています。 こちらのアトリエでは、作品制作、電子音響、ポートフォリオ、AIやMax/MSPを含む個別プロジェクト設計など、 より創作に踏み込んだ内容を扱います。
